## 🎯 Sobre Mim
Sou um estudante de Engenharia de Software baseado em Brasília, com foco absoluto em me tornar um Especialista Backend. 

Meu objetivo é conquistar minha primeira oportunidade como **Estagiário**, trazendo comigo uma base sólida de estudos, ética de trabalho e vontade de resolver problemas reais. Diferente de criar apenas "código que funciona", eu estudo para criar software **seguro, escalável e de fácil manutenção** — exatamente o que grandes instituições, bancos e órgãos públicos exigem.

- 📖 **Atualmente focado em:** Dominar o ecossistema Spring (Boot, Data JPA, Security) e modelagem de bancos de dados relacionais.
- 🧪 **O que estou aprendendo agora:** Testes Unitários com JUnit e Mockito (porque código sem teste é código legado).
- 💡 **Mentalidade:** Acredito que dominar os fundamentos (POO, SOLID) é mais importante do que saber a sintaxe do framework da moda.
